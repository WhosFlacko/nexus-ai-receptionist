## 🚀 Quick Deploy to Railway

1. Upload these files to your GitHub repo
2. Connect GitHub repo to Railway
3. Add environment variables in Railway:
   - TWILIO_ACCOUNT_SID
   -    - TWILIO_AUTH_TOKEN
   - OPENAI_API_KEY
4. Get Railway URL and configure Twilio webhook
